# Gypsy AI Chatbot 🤖

An AI-powered chatbot built using React and Vite.  
This project allows users to interact with an AI assistant in real time.

## 🚀 Features
- Real-time chat interface
- AI-powered responses
- Fast and responsive UI (Vite + React)
- Clean modern design

## 🛠️ Tech Stack
- React.js
- Vite
- Express JS
- TalwindCSS

## 📦 Installation

```bash
npm install
npm run dev
